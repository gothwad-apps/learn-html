Document Outline

<!DOCTYPE html> Declares the HTML version (this is HTML5)
<html> Root element of the entire HTML document

This encloses the entire page.

  • <html>
    • <head>... meta info
    • <body>... page content
<head> Contains meta-information about the page (not visible)

Includes title, stylesheet links, scripts, etc.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body> Contains the visible content of the page

All visible elements (headings, paragraphs, images, links, etc.) go here.

<body>
  <h1>Main Heading</h1>
  <p>This is a paragraph.</p>
  <img src="image.jpg" alt="Description">
</body>

Document Structure

<h1> - <h6> Headings (most to least important) 
<h1>Level 1 Heading</h1>
<h2>Level 2 Heading</h2>
<h3>Level 3 Heading</h3>
<!-- ... up to h6 -->

Helps structure the content.

<div> A block-level section (division)

Generic container for styling or grouping.

<div class="container">
  <h2>Section Title</h2>
  <p>Some content.</p>
</div>
<span> An inline section

Used to style a part of text.

<p>This is <span style="color: red;">red</span> text.</p>
<p> Paragraph 
<p>This is a paragraph of text. Browsers usually add some vertical space before and after paragraphs.</p>
<br> Line break (new line)

Forces a line break.

First line.<br>
Second line.
<hr> Horizontal rule (thematic break)

Represents a thematic break in content, often rendered as a horizontal line.

<p>First topic.</p>
<hr>
<p>Second topic.</p>

Form Elements

<form>Container for user input
<form action="/submit-page.php" method="post">
  <!-- Form elements go here -->
  <label for="username-id">Username:</label>
  <input type="text" id="username-id" name="username">

  <button type="submit">Submit</button>
</form>
<fieldset>Group related form elements
<form>
  <fieldset>
    <legend>Personal Information:</legend>
    <label for="fname">First name:</label><br>
    <input type="text" id="fname" name="fname"><br>
    <label for="lname">Last name:</label><br>
    <input type="text" id="lname" name="lname">
  </fieldset>
</form>
<legend>Caption for <fieldset>
<label>Label for form control (use `for` attribute)
<!-- Connects label to input via id -->
<label for="email-input">Email Address:</label>
<input type="email" id="email-input" name="email">
<input>Input field (many types)
<!-- Text Input -->
<label for="name-in">Name:</label>
<input type="text" id="name-in" name="name"><br>

<!-- Radio Button -->

<label for="html-radio">HTML</label><br>
<input type="radio" id="css-radio" name="fav_language" value="CSS">
<label for="css-radio">CSS</label><br>

<!-- Checkbox -->

<label for="vehicle1-check"> I have a bike</label><br>

<!-- Submit Button (using input) -->

<!-- Other types: password, email, number, date, etc. -->

Form Events (JavaScript)

These are JavaScript event handlers that can be attached to form elements to react to user interactions. Modern practice favors adding listeners via JavaScript (e.g., `element.addEventListener('change', ...)`), rather than inline attributes shown here for brevity.

onblurWhen element loses focus
<input type="text" onblur="myFunction()">

myFunction() runs when the user clicks outside the input field.

onchangeWhen element's value changes (and loses focus for text/textarea)
<select onchange="showSelection(this.value)">...</select>

showSelection() runs when the user selects a different option in the dropdown.

onfocusWhen element gains focus
<input type="text" onfocus="highlightField(this)">

highlightField() runs when the user clicks inside the input field.

onresetWhen form is reset
<form onreset="confirmReset()">
  ...
  <button type="reset">Reset Form</button>
</form>

confirmReset() runs when the reset button is clicked.

onselectWhen text is selected in input/textarea
<input type="text" onselect="logSelection()" value="Some text">

logSelection() runs when the user selects part of the text in this field.

onsubmitWhen form is submitted
<form onsubmit="return validateForm()">
  ...
  <button type="submit">Submit Form</button>
</form>

validateForm() runs before the form submits. If it returns false, submission is stopped.
